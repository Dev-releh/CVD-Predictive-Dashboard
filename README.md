Predictive Analytics Dashboard – CVD Risk Detection (Research Prototype)
📌 Project Overview
A research prototype for a predictive analytics dashboard designed to support early cardiovascular disease (CVD) risk detection in South African public hospitals. The project aimed to bridge the gap between machine learning models and practical clinical implementation. Completed as part of a Research Module at Central University of Technology.

📊 Key Highlights
🖼️ High-fidelity Figma prototype designed

📊 Quantitative analysis of cardiovascular datasets conducted

📋 User requirements gathered via questionnaires

🧩 Use case diagrams, ERD, and system architecture created

📄 Comprehensive research report produced

🛠️ Tools Used
Figma – High-fidelity interactive prototype

Python (Scikit-learn, Pandas) – Data analysis and model development (conceptual)

Google Forms – User requirements gathering

Draw.io – System architecture and use case diagrams
